# FIRE RUNNER 🔥

Fire Runner is a cross-platform multiplayer action game built with Unity.

## Features
- Player movement + mobile gyro aiming
- Weapons & attachments
- Zone shrinking system
- AI bots
- Multiplayer-ready architecture
- Firebase authentication
- PlayFab backend (rank & leaderboard)

## How to Use
1. Create a Unity 2022.3 LTS project
2. Copy this folder into Assets/
3. Install required packages (Photon Fusion, Firebase, PlayFab)
4. Open scene and press Play

## Platforms
- Android
- Windows PC
